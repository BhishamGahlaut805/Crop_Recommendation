Crop_Recommendation_Data
******************Crop Recommendation dataset analysis.****************

The dataset has 2200 entries for the columns like crop type, N, P, K level in the soil, temperature, humidity, ph value, rainfall. The graphs are plotted using Python libraries matplotlib, seaborn and plotted graphs including Nitrogen distribution across various crops, temperature vs humidity, average nutrients (N, P, k) by each crop(highlighting nutrients requirements).
